Accessibility

This website has been designed to be accessible to as many users as possible. This means that you should be able to:

  • zoom in up to 400% without the text spilling off the screen (WCAG 2.2 Success Criterion 1.4.4)
  • navigate most of the website using a keyboard or speech recognition software such as JAWS, NVDA and VoiceOver
  • listen to most of the website using a screen reader
  • be able to read text easily with visual impairments

These concerns have been addressed by:

  • using flexboxes and wrapping to ensure that text is always on-screen
  • used appropriate semantic elements
  • added suitable alt text to all images
  • ensured high contrast between text and their backgrounds (WCAG 2.2 Success Criterion 1.4.3)
  • used an accessible font (Atkinson Hyperlegible).

Should you have a disability that is not directly addressed by these considerations, AbilityNet has advice on how to make using your device easier.

Our site includes links to third-party providers. While we strive to make our site accessible, we cannot guarantee the accessibility of all external services and documentation.
